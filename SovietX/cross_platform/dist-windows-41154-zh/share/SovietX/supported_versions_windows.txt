# 如何获取版本信息：
#   1. 右键 Weixin.exe → 属性 → 详细信息

#      - 产品版本 或 内部版本号 = build_version
#   2. 或使用 GetFileVersionInfo API 读取
#   3. 在 Weixin.dll 中搜索版本字符串确认
#
# 逆向工具推荐：
#   - IDA Pro 7.x+ (商业) 或 Ghidra (免费, NSA 出品)
#   - x64dbg (免费, 动态调试)
#   - PE-bear / CFF Explorer (PE 头分析)

# □ hookPointerVA            - 撤回消息处理函数虚拟地址

# □ multiOpenPreventInstanceVA - 多开互斥检查
# □ groupExitDBApplyVA       - 退群 DB 操作
# □ openURLWebViewKindVA     - URL WebView 处理
# □ sendMsgCGIVA             - 发送消息 CGI 函数
# □ MessageWrap layout       - 消息包装结构体偏移量
#   - contentOffset

#   - createTimeMsOffset
